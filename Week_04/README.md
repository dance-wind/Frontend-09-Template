### 学习笔记
#### 字符串分析算法
+ 字典树
  - 大量高重复字符串的存储与分析
+ KMP
    - 在长字符里找模式（找部分字符串）
+ Wildcard
    - 带通配符的字符串模式
    - 只有两种通配符
    - 「?」表示匹配任意字符
    - 「*」表示任意数量的任意字符
    - 时间复杂度可以达到O(n)或者O(m+n)
+ 正则
    - 字符串通用模式匹配
+ 状态机
    - 通用的字符串分析
+ LLLR
    - 字符串多层级结构分析
